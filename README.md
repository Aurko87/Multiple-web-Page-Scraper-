# Web Scraping Projects

A collection of web scraping projects built with Python.

## Projects

### 1. Quotes Scraper
- Scrapes all 10 pages from quotes.toscrape.com
- Extracts quotes, authors, and tags
- Saves data to CSV

### 2. Books Scraper
- Scrapes all 50 pages from books.toscrape.com
- Extracts book title, price, and rating
- Saves 1000+ books to CSV

## Tech Stack
- Python
- Requests
- BeautifulSoup4
